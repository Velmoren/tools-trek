Atlassian cloud migration

Server/Data Center to Cloud Migration: This involves moving from a self-managed Atlassian infrastructure environment to Atlassian's Cloud services. It shifts the responsibility for system accessibility and updates to the provider, which enhances reliability, reduces maintenance workload, and ensures access to the latest features and security updates.

Cloud to Data Center Migration: For enterprises using Atlassian Data Center products, migrating from the Cloud can offer scalability and provide complete control by allowing data to be stored internally, which restricts physical access.

Server to Data Center Migration: This option is for organizations looking to scale their operations without moving to the cloud by upgrading from the outdated and unsupported Atlassian Server version to Atlassian Data Center.
